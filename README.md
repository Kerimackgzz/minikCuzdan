# Mini Dijital Cüzdan

Python ile yazılmış basit bir cüzdan uygulaması. Kullanıcı menü üzerinden 
para yatırma, çekme, bakiye görüntüleme ve işlem geçmişi inceleme 
işlemleri yapabilir.

## Özellikler

- Kullanıcı adı kaydetme ve kişisel karşılama
- Para yatırma (geçersiz miktar kontrolü ile)
- Para çekme (yetersiz bakiye ve geçersiz miktar kontrolü ile)
- Anlık bakiye görüntüleme
- İşlem geçmişi (tüm yapılan işlemlerin listesi)
- Hatalı input'a karşı koruma (try/except ile)
- Menü tabanlı kullanıcı arayüzü

## Kullanılan Kavramlar

- Fonksiyon tasarımı ve guard clause pattern
- Input validation (geçersiz değer kontrolü)
- `while` döngüsü ile menü sistemi
- Liste (list) veri yapısı ile işlem geçmişi
- `try/except` ile hata yakalama
- `isinstance()` ile tip kontrolü
- `continue` ile akış kontrolü
