# Mini Dijital Cüzdan

Python ile yazılmış basit bir cüzdan uygulaması. Kullanıcı menü üzerinden 
para yatırma, çekme ve bakiye görüntüleme işlemleri yapabilir.

## Özellikler

- Para yatırma (negatif miktar kontrolü ile)
- Para çekme (yetersiz bakiye kontrolü ile)
- Bakiye görüntüleme
- Menü tabanlı arayüz

## Kullanılan Kavramlar

- Fonksiyon tasarımı
- Input validation (geçersiz değer kontrolü)
- `while` döngüsü ile menü sistemi
- Guard clause pattern (early return)
